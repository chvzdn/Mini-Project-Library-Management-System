# Library Management System

A simple Python console-based Library Management System using Object-Oriented Programming (OOP).

## Features

- Add Book
- Register Member
- Borrow Book
- Return Book
- View Books
- View Members
- View Loans
- Exit Program

## Technologies Used

- Python 3
- OOP Concepts
- Dictionaries
- Exception Handling

## Project Structure

```text
library-management-system/
│
├── main.py
├── models/
├── services/
├── exceptions/
└── README.md
```

## How to Run

```bash
python main.py
```
